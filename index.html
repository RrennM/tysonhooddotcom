Bootstrap to WordPress

Tools used:

  • Custom Theme
  • CPT

About the project

This site is the project on which Brad Hussey's Bootstrap to WordPress course is based. The project starts out as a fully reponsive, static site built web-first with Bootstrap. It's then built for tablets, standard mobile, and smaller mobile screens. It features modals, parallax backgrounds, sprites, and other effects to give the site modern style.

The site is then made into a custom WordPress theme. A blank base theme is generated through Underscores (_s) and then the code from the static site is transformed into the WordPress theme. Along the way, the project used advanced custom fields, custom post types, and more, to make the once static site, a completely dynamic and user friendly experience.

About the project

This static website was the capstone project from the SkillCrush Career Blueprint. Using HTML5 and CSS3, I was to design and develop my very own personal website. I also used good design principles to create my own brand, business card and company logo(s). Some of the efforts which went into this site are complimentary typefacing and a analogous color scheme.

The site was build from the ground up. The process to do this was to first develop a wireframe. From there, I developed a mock up in Photoshop to get a feel for how it would all come together. I used a design tile to come up with possible colors and typefacing of which I'd like to use. Then onto final coding. It was written desktop-first, but is fully responsive to tablets and mobile devices.

In one of our lessons, something was said like, "A portfolio site is never finished. It's always evolving with any new skills." This website is a testament to that, as it has already come quite a ways from it's original version and my guess is it still has many changes in its future.
